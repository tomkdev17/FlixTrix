The name of this API is FlixTrix.

Users will be able to create/delete an account in which they'll be able to access information about movies, directors, genres, etc. as it pertains to the film world.

Here are some examples of the types of HTTP requests you could make with their corresponding URL endpoints

Business Logic URL HTTP Request Body Format Response Body Format
Return all Movies /movies GET None A JSON object holding all of the movies
Return data about a single movie by title /movies/:title GET None A JSON object holding all of the data concerning a single movie (Title, Director)
Return a description of a genre by name (i.e. Horror) /movies/genres/:genreName GET None A JSON object holding the description of a particular genre
Return information about a Director by name /movies/directors/:directorName GET None A JSON object holding data about a particular director (Bio, D.O.B., Filmography)
Allow new users to register an account /users POST A JSON object containing the new User information Example:
`{"Name": "John Doe", "Favorite Movies": []}`		 A JSON object reflecting the new User input, and also including a user ID
Allow users to update their account info (username) /users/:id PUT A JSON object containing the updated parameters Example:
`{"Name": "New John Doe", "Favorite Movies": []}`		 A JSON object reflecting the updated User parameters
Allow users to add a movie to their favortites list /users/:id/:movieTitle POST None A text message indicating that the movie has been added to the favorites list
Allow users to remove a movie from their favorites list /users/:id/:movieTitle DELETE None A text message indicating that the movie has been removed from the favorites list
Allow existing users to deregister their acocunt /users/:id DELETE None A text message indicating that the user's account information has been removed
